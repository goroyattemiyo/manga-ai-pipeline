## 1. コマ枠・構成の演出

### 回想・フラッシュバック
- **コマ枠の白黒反転**：通常は黒枠→回想は白枠（または枠なし白フチ）
- **枠の形を変える**：通常は四角→回想は角丸・楕円・雲形
- **空のコマ枠を重ねる**：現在シーンの上に小さい枠を浮かせて回想へ誘導
- **斜線・波線枠**：回想・夢・記憶の演出に使う
- **コマとコマの間に細い白帯**：時間の経過・切り替わりを示す
- **モノクロ or セピア調**：回想はフラットカラーを落として彩度を下げる（またはトーンのみ）

### 時間経過・場面転換
- **真っ黒or真っ白のコマ**：暗転・場面切り替え
- **背景のみコマ（キャラなし）**：時間帯・場所の説明
- **ページをまたぐ大ゴマ**：重要な場面・感情のピーク
- **細いコマ→大ゴマ**：緊張→開放の演出

### コマサイズの使い方
- **大ゴマ（1/2ページ以上）**：インパクト・登場・感情のピーク
- **小コマ連打**：テンポが速い・細かいリアクションの積み重ね
- **横長コマ**：広がり・パノラマ・空や宇宙
- **縦長コマ**：人物の全身・落下・上昇
- **コマ枠なし（フチなし）**：夢・イメージ・神秘的な場面

## 2. 効果線（集中線・放射線・速度線）

### 集中線（放射状）
- **中心に向かって集まる線**：視線を中央に誘導・衝撃・発見・驚き
- **用途**：重要アイテムの登場・キャラの決めポーズ・突然の登場

### 放射線（外に広がる）
- **中心から外へ広がる線**：爆発・感情の爆発・驚愕
- **用途**：「えっ！？」「なんだと！？」の驚きシーン

### 速度線（流線）
- **横方向の平行線**：左右への移動・スピード感
- **縦方向の平行線**：落下・上昇
- **用途**：走る・飛ぶ・乗り物の移動・猪突猛進なキャラの動き

### 斜め効果線
- **斜め45度の線群**：緊張・圧迫感・シリアスな空気
- **用途**：重要キャラの登場・不気味な場所・緊迫シーン

## 3. 感情・状態を表すマンガ記号

### 汗・冷や汗
- **大粒の汗ひとつ**：困惑・呆れ・緊張
- **汗がたくさん飛ぶ**：パニック・焦り
- **冷や汗（縦線＋汗）**：冷や汗・恐怖

### 怒りマーク
- **こめかみの「ギギギ」マーク**：怒り・苛立ち
- **大きいほど激しい怒り**

### ショック・落胆
- **縦線（暗い影）が顔に走る**：落胆・絶望
- **目が点になる**：軽い驚き・ポカン
- **白目になる**：強い衝撃・気絶寸前

### ときめき・興奮
- **ハートやキラキラ（☆）が飛ぶ**：ときめき・興奮・感動
- **目がキラキラ（スクリーントーンのキラ）**：憧れ・感動

### 疑問・混乱
- **「？」マーク**：疑問
- **「！」マーク**：気づき・驚き
- **「！？」同時**：混乱・衝撃と疑問が同時

### 沈黙・静寂
- **「…」（三点リーダー）**：無言・間・沈黙
- **コマ全体が静かなトーン**：静寂の演出

## 4. 背景トーン・テクスチャ

### スクリーントーン（アナログ）／フラットカラー（デジタル）
- **点描トーン（網点）**：影・グレー表現
- **線トーン**：スピード・テクスチャ
- **砂目トーン**：ざらっとした質感・不安感
- **花柄・ハート柄**：ときめき・乙女心の演出背景

### 感情背景（漫画特有）
- **バラやキラキラが飛ぶ背景**：ときめき・ドヤ感
- **暗い縦線背景**：落胆・絶望・ショック
- **爆発背景（ギザギザ放射）**：驚き・怒り・衝撃
- **白一色（フラッシュ）**：衝撃の瞬間・強い感情のピーク
- **星空・宇宙背景**：神秘・壮大・重要キャラ登場シーン

### シーン別・推奨背景の考え方
- **日常シーン**：シンプルなフラットカラー室内・過度に写実的にしない
- **神秘・異世界シーン**：星空・光・霧・靄のテクスチャ
- **緊迫・禁止エリア**：暗い斜め効果線＋霧・靄のテクスチャ
- **重要キャラ登場**：バックライト＋放射線or斜め効果線

## 5. フキダシ（セリフ枠）の種類

### 通常フキダシ
- **丸・楕円**：普通の会話
- **角丸四角**：落ち着いたセリフ・ナレーション

### 感情フキダシ
- **ギザギザ枠（爆発型）**：叫び・怒り・驚き
- **震える枠**：恐怖・緊張・怒り
- **二重丸枠**：強調・重要なセリフ

### 特殊フキダシ
- **雲形（モコモコ）**：心の声・内心・独り言
- **細い枠・小さいフキダシ**：小声・つぶやき・無口キャラのセリフに最適
- **枠なし・流れ字**：ナレーション・説明文
- **角ばった枠**：機械音・通信音・電話・ホログラム通話のセリフ

### 遠くから届く声・読者向け独り言の推奨フキダシ
- **枠なし or 細い楕円**：遠くから届く感じ・読者だけに届く演出
- **フキダシに尻尾なし**：どこからともなく聞こえる感じ
- **ページ端に小さく配置**：主要な会話と区別する

## 6. 擬音（オノマトペ）

### 動作系
- **ドン・バン**：衝突・扉が開く・強い動作
- **ふわ・ふらっ**：浮遊・ぼんやり
- **ぴたっ**：突然止まる・固まる
- **ぷるぷる**：震え・怒り・緊張
- **どかっ・ズサッ**：転倒・激しい動き

### 感情系
- **ドキッ**：驚き・ときめき
- **ズーン**：落胆・重い気持ち
- **キラーン**：ひらめき・目が光る
- **カッ（白フラッシュ）**：強い衝撃・登場の演出

### 環境系
- **シーン**：静寂
- **ざわ…ざわ…**：不穏な空気・群衆のざわめき
- **ゴゴゴ…**：迫力・威圧・重要キャラの登場

## 7. キャラクタータイプ別の演出

### 猪突猛進タイプ（主人公が突き進む）
「好奇心に駆られてまわりが見えなくなる」を表現する技法：

- **速度線を体全体に走らせる**：突進・勢い
- **背景を流す（ブラー）**：スピード感
- **周囲のキャラが呆れ顔で取り残される**：主人公だけ前に進んでいるコマ
- **小さいコマを連打**：テンポよく動きを積み重ねる
- **セリフが行動の後に来る**：しゃべりながら走っている・動きながら決断している
- **コマ枠を突き破る体**：枠からはみ出すことでエネルギー感を演出
- **サブキャラが置いてきぼりになるコマ**：主人公が走り去った後にサブキャラが一人残っている

### 無口・リアクションタイプ（サブキャラ）
「セリフなしで感情を伝える」技法：

- **汗ひとつ・じと目・首かしげ**：困惑・呆れを小さく表現
- **一歩引く・無言で指さす**：静かなツッコミ
- **ぷるぷる震える・怒りマーク**：感情の高ぶりを記号で表現
- **背景に小さな効果線**：さりげない反応の強調
- **セリフなしコマを連続させてオチで一言刺す**：間でギャグを作る

### 神秘・権威タイプ（重要キャラ）
「存在感を出しつつ全貌を見せない」技法：

- **大ゴマ＋放射線**：登場の格を出す
- **シルエット・輪郭のみで描く**：神秘性を保つ
- **枠なしフキダシ・尻尾なし**：どこからともなく届く声の演出
- **ゴゴゴ擬音**：威圧感・迫力
- **頻度を絞る**：毎回出すと神秘性が薄れる

## 8. 回想切り替え演出・完全整理

### カテゴリ1：コマ枠の変化

| 手法 | 見た目 | 使われ方 |
|---|---|---|
| **白枠反転** | 通常の黒枠→白枠に変わる | 最もスタンダード。少年・少女漫画どちらでも使われる |
| **波線枠** | コマの枠が波打っている | 夢・記憶・おぼろげな回想に使う |
| **破線枠** | 枠が点線になっている | 断片的な記憶・不確かな回想 |
| **枠なし（フチなし）** | コマの境界線がない | 神秘的・感情的なシーン |
| **角丸枠** | 四角の角が丸くなっている | やわらかい記憶・穏やかな回想 |
| **二重枠** | 枠が二重線になっている | 強調・重要な回想シーン |

### カテゴリ2：切り替え用の専用コマ

| 手法 | 見た目 | 使われ方 |
|---|---|---|
| **白コマ1枚** | 真っ白のコマを1枚挟む | フラッシュ・記憶が蘇る瞬間 |
| **黒コマ1枚** | 真っ黒のコマを1枚挟む | 暗転・重い記憶への移行 |
| **背景のみコマ** | キャラなし・場所だけ | 「あの場所」を示してから回想へ |
| **重ねコマ** | 現在コマの中に小さい回想コマが浮かぶ | 「思い出している」感が視覚的に出る |
| **フェードコマ** | 現在→ぼかし→回想とグラデーション | デジタル漫画でよく使われる |

### カテゴリ3：エフェクト・トーン

| 手法 | 見た目 | 使われ方 |
|---|---|---|
| **集中線（回想方向へ）** | キャラの顔を中心に線が集まる | 「思い出した瞬間」の衝撃を表現 |
| **放射線（回想から広がる）** | 回想コマの背景に放射線 | 回想シーンの始まりを印象づける |
| **砂目トーン** | ざらっとしたテクスチャ | 古い記憶・不鮮明な過去 |
| **スクリーントーン全面貼り** | コマ全体に均一なトーン | 回想中であることを継続して示す |
| **セピア調（デジタル）** | 彩度を落とした茶色っぽい色味 | 過去感・懐かしさ |
| **モノクロ化** | 回想中はカラーなし | 現在との対比を強調 |

### カテゴリ4：ナレーション・文字演出

| 手法 | 見た目 | 使われ方 |
|---|---|---|
| **「あのとき――」枠** | 長方形のナレーション枠 | 時制の変化を言葉で明示 |
| **「X年前」テキスト** | コマ上部にテキスト | 時間の距離を数字で示す |
| **「少し前のこと」** | ナレーション枠 | 近い過去への回想 |
| **斜め文字・手書き風文字** | 傾いたフォント | 記憶・日記・回想らしさを演出 |

### カテゴリ5：フキダシの変化（回想中）

| 手法 | 見た目 | 使われ方 |
|---|---|---|
| **角丸フキダシに統一** | 通常の丸フキダシ→角丸に | 回想中のセリフと現在のセリフを区別 |
| **薄いフキダシ** | フキダシの枠線が細い・薄い | 遠い記憶・おぼろげな声 |
| **括弧書き** | 「（セリフ）」 | 心の中での回想・声として聞こえない |

## 9. 回想シーンの演出パターン（標準構成）

「現在→回想→現在」の構成を使う場合の推奨演出：

### 回想への入り方
- **現在シーンの最後のコマ**：キャラが何かを思い出す表情
- **重ねコマ**：現在コマの中に小さい白枠の回想コマを浮かせる
- **「あのとき――」などのナレーション枠**：時制の切り替えを明示
- **白コマ1枚**を挟む：時間軸の切り替えを視覚化

### 回想中の演出
- **コマ枠を白枠（反転）**：回想中であることを継続して示す
- **彩度を落とす or トーンのみ**：過去感を出す（デジタルならセピアフィルタ）
- **フキダシを角丸に統一**：回想内のセリフと現在のセリフを区別

### 回想からの戻り方
- **黒コマ1枚**：回想終了の暗転
- **コマ枠を黒に戻す**：現在に戻ったことを示す
- **現在のキャラの表情コマ**：回想から目覚めたような顔

### 推奨の組み合わせ（シンプル版）

```
現在コマ（通常黒枠）
　↓
重ねコマ：現在コマの中に小さい白枠の回想コマが浮かぶ
　↓
白コマ1枚 ＋ ナレーション「―― 少し前のこと。」
　↓
回想中：白枠 ＋ スクリーントーン全面（または彩度落とし）
　↓
黒コマ1枚（回想終了の暗転）
　↓
通常黒枠（現在に戻る）
```

選択理由：
- 重ねコマで「キャラが思い出している」感を視覚化
- 白枠で回想中であることを継続表示
- 黒コマで明確に回想を終わらせる
- シンプルで読者に伝わりやすく、テンポを崩さない

## 10. シーン別・推奨演出クイックリファレンス

| シーン | 推奨効果 |
|---|---|
| 冒頭・現在シーン | 通常コマ・フラットカラー |
| 回想への切り替え | 重ねコマ→白コマ1枚→白枠コマ枠 |
| 広い場所・移動シーン | 横長コマ・パノラマ背景 |
| 禁止エリア・緊迫場所 | 斜め効果線・暗い背景・ざわ…演出 |
| 主人公の猪突猛進 | 速度線・コマ枠突き破り・サブキャラ置いてきぼり |
| 重要キャラの登場 | 大ゴマ・放射線・ゴゴゴ擬音・枠なしフキダシ |
| 主人公が興奮している | 目がキラキラ・ハートが飛ぶ |
| 神秘的な誕生・変化 | コマ枠なし・幻想背景・キラキラトーン |
| 回想から現在に戻る | 黒コマ1枚→通常黒枠に戻る |
| オチ | 小コマ連打→大ゴマでオチ |
| 遠くからのツッコミ | 枠なしフキダシ・尻尾なし・ページ端に小さく |
