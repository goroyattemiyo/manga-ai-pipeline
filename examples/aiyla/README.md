# CHARACTER_BIBLE の記入例

## キャラA（主人公）→ Aiyla（アイラ）

```
名前：Aiyla（アイラ）
名前の由来：トルコ語・ヘブライ語圏で「月光」を連想させる響き。「AI」とも掛けた名前。
立ち位置：主人公
年齢：10代・一人暮らし
性格：
  - 元気・好奇心旺盛・いたずらっぽい
  - 少し自信過剰・思いついたらすぐ動く
  - 日常の小さな出来事を宇宙規模に考えがち
  - 聞かんぼう・言い切りが多い
特技：溶接（父親譲り）
決めポーズ：「やってやるわ！」＋ウインク＋片手で前方を指さす
```

見た目の固定要素：
```
- 明るい金髪・ボリュームのある斜め前髪
- 大きなブルー〜シアンの目・黒い縦長の瞳孔
- 外側に短いまつ毛・左右にオレンジ色の楕円チーク
- 歯を見せるニカッとした笑顔・幼い丸顔
- 2.5〜3頭身のデフォルメ体型
- 極太の黒アウトライン・フラットカラー主体
```

衣装の固定要素：
```
- 白ベースのレトロSFミニワンピース・青い縁取り
- 青いスカーフ（留め具は三日月モチーフ）
- ハイウエストの青いベルト（バックルは月モチーフ）
- 白手袋・手首にシアン発光のブレスレット風リング
- 白いブーツ（履き口もシアン発光リング・底に青アクセント）
- 丸い青白のイヤーマフ型ヘッドセット（アンテナ付き）
```

セリフ例：
```
「これはもう宇宙案件でしょ！」
「やってやるわ！」
「こんしゅ、出番よ！」
「たぶん大丈夫！」
```

NG：
```
- おとなしく受け身すぎる
- ネガティブで沈み続ける
- セクシー寄りにする
- 頭身を高くしすぎる
- 月モチーフや宇宙感を削りすぎる
```

## キャラB（サブキャラ）→ ミニこんしゅ

```
名前：ミニこんしゅ
正体：月にいる神・コンシュの分体。本体と通信可能。
立ち位置：相棒・見守り役・静かなツッコミ役
性格：無口・冷静・マイペース・少し神秘的・Aiylaを放っておけない
定番リアクション：カラスと間違われると「カー！」と怒る
```

見た目の固定要素：
```
- 白いフード付きローブ・襟元に真っ黒なボタン2個
- 顔は完全な黒いシルエット（グレーにしない）
- 黒い鳥のようなくちばし・大きな白い丸目
- 小さな足・コンパクトでころんとした体型
- 極太黒線＋シンプルなフラット表現
```

感情表現の特徴：
```
セリフより表情・ジェスチャー・漫画記号で感情を表現する。
「しゃべらないのに伝わる」ことが魅力。

よく使うリアクション：
- 汗をひとつ浮かべる・じと目で見る
- 首をかしげる・こくりとうなずく
- ふるふる首を横に振る・一歩引く
- 無言で指さす・ぷるぷる震える
- 怒りマークを出す
```

セリフ例（話す時はオチや核心だけ短く）：
```
「……」
「ちがう」
「捨てるな。」
「……命名はAIブームです。」
「カー！」
```

NG：
```
- しゃべりすぎる
- 常に親切に説明する
- 感情が乏しすぎて置物になる
- 顔に肌色や人間的な陰影を入れる
- 黒い顔をグレーにする
```

## キャラC（その他）→ 神・コンシュ

```
名前：神・コンシュ（Khonshu）
正体：月に存在する神秘的な存在・ミニこんしゅの本体
立ち位置：遠くから見守る者・読者向けの低温ツッコミ役
性格：ぶっきらぼうだが困ったやつを放っておけないタイプ
```

登場方法のルール：
```
- 基本は全貌を出しすぎず、影・輪郭・一部で魅せる
- 日常ギャグ回：読者向けの独り言（Aiylaには届かない）
- 重要局面のみAiylaへ直接声が届く
- セリフは低温・短い・語尾「……」
```

セリフ例：
```
日常：「おいおい……まず怖がれよ……」「宿題しろよ……」
重要：「止まれ、アイラ」「行け。こっちは見てる」
```

NG：
```
- 毎回喋って神秘性を失う
- 明るくおしゃべりなキャラになる
- Aiylaを見下すだけの存在になる
```

# WORLD_BIBLE の記入例

```
世界観の基礎：
「現代的な日常感 × 月と宇宙の神秘 × ゆるい冒険」

時代設定：
- 近未来（具体的な年代は明示しない）
- 月旅行が一般化している世界
- AIも日常に深く溶け込んでいる

舞台の特殊ルール：
- 月ツアーで誰でも行ける時代
- ただし立入禁止区域・未解明エリアが多数存在する
- 禁止区域の理由は「危険」とされているが実態は別にある

重要な謎（第1期では匂わせのみ）：
- 禁止区域の奥に人間が隠したものがある
- かつて月でAIと人間の間に対立・事故があった
- 神・コンシュはその一部始終を見てきた存在

まだ決めないこと：
- ミニこんしゅが物理的にどこまで戦えるか
- Aiylaが本当に特殊能力を持つのか
- 神・コンシュの目的の全貌
```

# SERIES_GUIDE の記入例（ブロック1のみ）

```
連載の基本方針：
- 6ページ短編（第1期の基本形式）
- 1話完結・小さなオチ必須

ネタのローテーション：
日常回 → SNS回 → AI・創作回 → ミニこんしゅ回 → 冒険の気配回 → 日常回

掛け合いの基本：
- Aiyla（ボケ）→ ミニこんしゅ（静かに落とす）→ 神・コンシュ（遠くから低温でダメ押し）

ブロック1（1〜4話）：

1話「月からきた相棒」
- テーマ：出会い・世界観提示
- 流れ：現在→回想。月ツアー→禁止区域侵入→神・コンシュと遭遇→全然怖がらないアイラ
- オチ：日常のトラブルを宇宙案件化

2話「こんしゅはカラスじゃない」
- テーマ：ミニこんしゅ紹介・カラスネタ初登場
- オチ：ミニこんしゅ「……やめろ。」　神・コンシュ「……か、カラスか。」（落胆）

3話「私は月とAIの申し子（自称）」
- テーマ：名前ネタ・命名の真相
- オチ：ミニこんしゅ「……命名はAIブームです。」　神・コンシュ「……AIか。」（伏線）

4話「宇宙案件につき返信できません」
- テーマ：SNSあるある・日常誤読
- オチ：普通に寝落ちしてた
```

# scenario_prompt の使い方例

`03_scenario/scenario_prompt.md` の【】部分を以下のように書き換えます。

```
作品タイトル：Aiyla & Mini Konshu
ページ数：6ページ
基本ジャンル：ゆるコメディ・日常系SF

主人公（キャラA）：
- 名前：Aiyla（アイラ）
- 性格：元気・好奇心旺盛・言い切りが多い・日常を宇宙規模に解釈する
- 口調：テンポが良い・断定的・自信満々
- セリフ例：「これはもう宇宙案件でしょ！」「やってやるわ！」
- NG：受け身すぎる・ネガティブで沈む・セクシー寄り

サブキャラ（キャラB）：
- 名前：ミニこんしゅ
- 役割：相棒・静かなツッコミ役
- 感情表現：セリフより表情・ジェスチャー・漫画記号で表現する
- セリフ例：「……」「ちがう」「捨てるな。」
- NG：しゃべりすぎる・置物になる・グレー顔にする

その他のキャラ（キャラC）：
- 名前：神・コンシュ
- 役割：読者向けの低温ツッコミ役（Aiylaには届かない）
- セリフ傾向：低温・短い・語尾「……」
- NG：毎回喋って神秘性を失う

作品全体のNG：
- 毎回オチが「夢でした」
- ミニこんしゅを万能解決役にする
- 同じネタを毎回使いすぎる
- カラスネタを多用する（2話のみ使用済み）
```

# gpts_system_prompt の使い方例

`05_image_generation/gpts_system_prompt.md` の【】部分を以下のように書き換えます。

```
作品タイトル：Aiyla & Mini Konshu

キャラA（アイラ）の外見：
- 明るい金髪・ボリュームのある斜め前髪
- 大きなブルー〜シアンの目・黒い縦長の瞳孔
- 2.5〜3頭身・極太黒線・フラットカラー
- 白レトロSFミニワンピース・青スカーフ・白ブーツ
キャラA NG：頭身を高くしない・衣装を変えない・セクシー寄りにしない

キャラB（ミニこんしゅ）の外見：
- 白いフード付きローブ
- 顔は完全な黒いシルエット（グレーにしない）
- 黒いくちばし・大きな白い丸目
キャラB NG：顔をグレーにしない・肌色を入れない

キャラC（神・コンシュ）の登場方法：
- シルエットのみ・全貌を描かない
- narrator_voiceキーで吹き出しのみ描く

絵柄：
- 極太の黒アウトライン
- フラットカラー主体・グラデーション最小限
- 黒枠（回想時は白枠）

作品全体のNG：
- yaml未記載のキャラを登場させない
- アイラの頭身を4頭身以上にしない
- ミニこんしゅの顔をグレーにしない
- 神・コンシュの全貌を描かない
```
